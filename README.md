# Rödpäls-Resan 🐕✨

En trygg och pedagogisk guide för ägare av **Finsk Spets**. Följ hundens utveckling från den första dagen som valp till ett tryggt vuxenliv.

## 🌟 Appens Innehåll

- **Utvecklingsresan:** Vecka-för-vecka-guide genom valpstadiet, tandömsning, slyngelålder och vuxenlivet.
- **Varningar & Säkerhet:** Färgkodad lista över farliga livsmedel och föremål med tydliga åtgärdsplaner.
- **Barn & Hund:** Gyllene regler och tips för hur barn och hund skapar en trygg relation.
- **Lugnande Råd:** Pedagogiska förklaringar av hundens beteende för att minska stress hos nya ägare.
- **Portabel:** Körs som en enda fil (`index.html`) – ingen installation eller server krävs.

---

## 🚀 Kom igång

### För hundägaren (Använd appen)
1. Ladda ner filen `dist/index.html`.
2. Dubbelklicka på filen för att öppna den i din webbläsare.

### För utvecklare
```bash
# Klona arkivet
git clone https://github.com/ChipChop87/R-dpels-Resan.git

# Installera och kör
cd R-dpels-Resan
npm install
npm run dev

# Bygg standalone-fil
npm run build
```
